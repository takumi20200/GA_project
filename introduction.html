في عالم اليوم الرقمي، أصبح تصميم المواقع مهارة أساسية في تطوير الحلول الرقمية الحديثة، إذ تلعب واجهات المستخدم دورًا رئيسيًا في جذب المستخدمين وتقديم تجربة تفاعلية وسهلة الاستخدام. يهدف هذا المشروع إلى تصميم وبناء واجهة موقع (Front-end)باستخدام HTML وCSS ، مما يتيح فهم الأساسيات دون الاعتماد على أطر عمل معقدة. من خلال هذا المشروع، تمكنا من استكشاف كيفية تصميم واجهات متجاوبة وجذابة بصريًا، مع مراعاة التوافق عبر مختلف الأجهزة. يساعد المشروع أيضًا على تنمية المهارات في تنظيم الأكواد وفهم أساليب التصميم الحديثة، مما يشكل أساسًا قويًا لمشاريع برمجية مستقبلية. .1 الهدف من المشروع يهدف المشروع إلى إنشاء موقع إلكتروني باستخدام تقنيات HTML وCSS فقط، مع دمج نافذة منبثقة بواسطة JavaScript. توفر النافذة المنبثقة تجربة مستخدم مميزة، حيث يمكن استخدامها في العديد من السيناريوهات، مثل عرض إشعارات مهمة، أو معلومات إضافية عند الضغط على زر معين. ويأتي هذا المشروع كخطوة أولى نحو إتقان تقنيات تطوير الويب البسيطة وإضافة عناصر تفاعلية باستخدام JavaScript، مما يُضفي على الموقع لمسة مهنية عالية الجودة. .2 الأدوات المستخدمة HTML: لإنشاء الهيكل الأساسي للموقع وتحديد أقسامه. CSS: لتصميم وتنسيق الموقع، وتحديد الألوان، الخطوط، والمسافات بين العناصر. Visual Studio Code: كمحرر رئيسي لتطوير الموقع، حيث يتيح العمل بسهولة مع ملفات HTML وCSS. Live Server: إضافة لـ Visual Studio Code لتمكين معاينة الموقع في الوقت الفعلي على المتصفح، مما يسمح بتحديث الصفحة تلقائيًا عند إجراء تعديلات على الكود. GitHub: ستستخدم GitHub لاحقًا لنشر الموقع واستضافته بشكل مجاني، مما يجعل الموقع متاحًا للجمهور عبر الإنترنت. .3 طريقة العمل التخطيط الأولي: بدأ المشروع بتحديد هيكل الموقع وتحديد الأقسام الرئيسية، مثل العنوان، التعريف، الأقسام الفرعية، وقسم البحث العلمي. كتابة الكود: HTML: تم إنشاء الهيكل الأساسي للموقع باستخدام عناصر HTML، مثل

CSS: تم تصميم وتنسيق الموقع باستخدام CSS، بما في ذلك إضافة ألوان، وتحديد محاذاة النصوص، وضبط المسافات بين العناصر، وإضافة تنسيقات تفاعلية لجعل الموقع جاذبًا. المعاينة والتعديل: تم استخدام Live Server لمعاينة الموقع بشكل مباشر على المتصفح، مما سمح بإجراء التعديلات في الوقت الفعلي وتصحيح الأخطاء بسرعة. الاستضافة: بعد الانتهاء من التصميم، سيتم رفع المشروع على GitHub، واستخدام GitHub Pages لاستضافة الموقع مجانًا. تتيح هذه الخدمة من GitHub نشر الموقع بشكل مباشر على الويب، مما يتيح لأي مستخدم الوصول إليه بسهولة عبر رابط عام. وتعد GitHub Pages حلاً ملائمًا لاستضافة المشاريع الصغيرة وصفحات الويب البسيطة، وتوفر ميزة التحديث التلقائي عند إجراء أي تعديلات على مستودع المشروع، مما يسهل عملية تطوير الموقع وإدارة الشيفرة بشكل مستمر. .4 التحديات ضبط التنسيق عبر مختلف المتصفحات: كان من التحديات التأكد من أن الموقع يظهر بالشكل الصحيح على جميع المتصفحات (مثل Chrome وFirefox). التنسيق والتحكم في التخطيط: احتاج المشروع إلى بعض التعديلات الدقيقة لضبط المحاذاة وتحديد المسافات بشكل مناسب بين العناصر وكذلك جعله يظهر من اليمين إلى اليسار حتى يتوافق مع لغتنا العربية. استخدام CSS بدون أدوات إضافية: الالتزام بتصميم الموقع باستخدام CSS فقط دون الاعتماد على أطر عمل مثل Bootstrap جعل العمل أصعب ولكن أضاف أيضًا تحديًا ممتعًا لتصميم الموقع من الصفر. التوافق مع أحجام الشاشات: كان تحديًا كبيرًا جعل الموقع متوافقًا مع مختلف أحجام الشاشات، بما في ذلك الهواتف المحمولة. .5 النتائج تم إنشاء موقع بسيط وجذاب باستخدام HTML وCSS، مع هيكل سهل التصفح وتصميم متناسق. عرض المشروع على المتصفح باستخدام Live Server ساهم في تسريع عملية التطوير، حيث كان من الممكن مشاهدة التعديلات مباشرة. بعد نشر المشروع على GitHub Pages، أصبح الموقع متاحًا للجمهور مع رابط خاص يمكن مشاركته مع الآخرين، مما يساهم في تسهيل وصول المستخدمين للموقع. الخاتمة: في الختام، قدم هذا المشروع تجربة تعليمية غنية، حيث ساعدنا على تطوير مهاراتنا في تصميم الواجهات باستخدام HTML وCSS وقليل من JavaScript. كانت العملية مليئة بالتحديات، خاصة في محاولة تحقيق التوافق عبر الأجهزة المختلفة، وقد تعلمنا الكثير عن أهمية التنظيم والدقة في بناء المواقع. من خلال البحث وتجربة حلول مختلفة، أصبحنا أكثر ثقة في القدرة على تقديم مواقع متجاوبة وفعّالة. آمل أن نكون قد حققنا الهدف المرجو من المشروع، وأن نساهم بهذه المعرفة في تطوير مشاريع قادمة. المراجع: YouTube - مصدر رئيسي للفيديوهات التعليمية حول أساسيات HTML وCSS، ساعدني في فهم الخطوات العملية لبناء وتصميم المواقع. Mozilla Developer Network (MDN) - أحد أهم المراجع في تطوير الويب، حيث وفرت توثيقًا شاملاً للأكواد وأفضل الممارسات في HTML وCSS. Codezilla - مصدر قيّم للحصول على الأكواد التوضيحية والنصائح العملية في تطوير صفحات ويب متجاوبة وصديقة للمستخدم. Chatgpt: استعملناه لشرح غير المفهوم وكذلك وللمساعدة في كتابة الكود والمشروع وشرحه
